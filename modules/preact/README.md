## Setup

Use the `preact-cli` to setup a new project from the template [widget-typescript](https://github.com/preactjs-templates/widget-typescript)
```bash
cd modules/preact
npx preact-cli create widget-typescript my-widget
cd my-widget
npm install
npm run dev
```
